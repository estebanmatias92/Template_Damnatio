Damnatio HTML5 & CSS3 Template.

---

Template Made entirely in HTML5 and CSS3, a proof that 
it is completely possible use the more advanced concepts 
of these technologies in current browsers, of course, except IE. 
Despite this latter have total and complete support in all other browsers.

Features:

- Full design on CSS3.
- HTML5 document structure.
- Roles and interaction with jQuery.
- Optimized loading rate (CDN usage, and images optimized).
- Nivo-Slider, personal modified.
- HTML5 Contact Form.
- Less for CSS preprocessing.
